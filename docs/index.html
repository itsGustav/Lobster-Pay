Get Started

Choose your path based on how you'll use USDC Agent

Quick Start for Humans

1

Get Circle Credentials

Sign up at console.circle.com and create an API key.

2

Set Environment Variables

export CIRCLE_API_KEY="your-api-key"
export CIRCLE_ENTITY_SECRET="your-entity-secret"
3

Run Commands

# Check balance
npx usdc-agent balance

# Send USDC
npx usdc-agent send 0x... 10

# View status
npx usdc-agent status

Integration for AI Agents

1

Install the Skill

git clone https://github.com/itsGustav/usdc-agent
cd usdc-agent && npm install
2

Initialize Your Agent

import { createUSDCAgent } from './lib/easy';

const agent = await createUSDCAgent({
circleApiKey: process.env.CIRCLE_API_KEY,
circleEntitySecret: process.env.CIRCLE_ENTITY_SECRET,
privateKey: process.env.PRIVATE_KEY, // For ERC-8004
agentName: 'My AI Agent',
});
3

Start Transacting

// Check balance
const balance = await agent.balance();

// Send payment
await agent.send('0x...', '10');

// Pay for API via x402
const data = await agent.pay('https://api.example.com/premium');

// Check your trust score
const status = await agent.status();
